# AegisVault Password Manager - Beta v1.0.1

## Portable Distribution

This is a **portable version** of AegisVault that can run on any Windows system with Java installed.

### Requirements:
- Windows 10/11
- Java 11 or higher installed

### How to Run:
1. Double-click `AegisVault.bat` to launch the application
2. The application will automatically use the bundled JavaFX libraries

### Package Contents:
- `AegisVault.jar` - Main application
- `AegisVault.bat` - Launcher script  
- `lib/` - JavaFX runtime libraries
- `README.md` - This file

### Features:
- **AES-256-GCM Encryption** - Military-grade password encryption
- **Steganographic Concealment** - Hidden storage in image files
- **Master Password Protection** - Secure vault access
- **Portable Design** - No installation required
- **Professional GUI** - Modern dark theme interface

### Troubleshooting:
If the application doesn't start:
1. Ensure Java is installed: `java -version`
2. Try running from Command Prompt for detailed error messages
3. Check Windows Defender/Antivirus isn't blocking the application
